# İlk Gerçek Test Protokolü

## Katılımcı Sayısı

İlk deneme: 5 kişi  
Genişletilmiş deneme: 20 kişi

## Kişi Başına Görüntü

Başlangıç için:

- Aynı parmak
- En az 8 görüntü
- Mümkünse iki farklı oturum

## Tutulacak Alanlar

- Katılımcı kodu
- Parmak
- Görüntü numarası
- Kamera-parmak mesafesi
- LED-parmak mesafesi
- LED dalga boyu
- LED şiddeti/duty cycle
- Ortam ışığı notu
- Görüntü kalite notu
- Eşleşme skoru

## Kural

Gerçek isim veya gereksiz kişisel veri tutulmaz. Katılımcı kodu kullanılır. Gönüllü onam metni okunmadan görüntü alınmaz.

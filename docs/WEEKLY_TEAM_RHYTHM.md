# Haftalık Takip Ritmi

## Amaç

AI ekip görevleri GitHub Issues üzerinden kayıtlı ilerler. Her görev, issue yorumu veya commit ile izlenebilir olmalıdır.

## Ritim

Her çalışma bloğunda:

1. Ana sprint issue'su kontrol edilir.
2. Rol issue'larında açık checkbox'lar incelenir.
3. Yapılan iş ilgili issue'ya yorum olarak eklenir.
4. Tamamlanan checkbox işaretlenir.
5. Kod veya doküman değişikliği commit/push edilir.

## Kapanış Kuralı

Bir görev ancak şu üç şey varsa tamamlandı sayılır:

- GitHub issue yorumu
- Dosya/komut/test çıktısı
- Checkbox güncellemesi
